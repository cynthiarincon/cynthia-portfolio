    Natours Project

    Project Overview

    Natours is a fictional nature tour company landing page I built following Jonas Schmedtmann's "Advanced CSS and Sass" Udemy course. Through this code-along, I learned modern CSS techniques including animations, responsive design, and the BEM methodology.

    Technologies: HTML5, CSS3, Sass, BEM

    Key Techniques Learned

    BEM Methodology: Block-Element-Modifier naming conventions (.btn--white, .card__side)

    CSS Animations: Keyframe animations for smooth text and button effects (@keyframes moveInLeft)

    Clip-Path: Creating unique geometric shapes for modern design

    Sass Features: Variables, nesting, mixins, and partials for organized CSS

    Responsive Design: Mobile-first approach with media queries

    Advanced Selectors: Pseudo-elements (::after, ::before) for hover effects
